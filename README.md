# -# 多功能连点器（反检测增强版）

一款功能强大的自动化工具，支持连点、鼠标操作录制与回放、可视化流程编辑（含OCR、图像匹配、颜色检测、条件跳转）以及智能反检测机制，有效模拟人类操作，降低被检测风险。

## ✨ 功能特性

- **连点器**：自定义点击类型（左键/右键/中键/双击）和频率，支持反检测（时间+位置随机抖动）。
- **操作录制与回放**：录制鼠标移动和点击，可回放并调整速度倍数，支持绝对/相对坐标模式。
- **点击流程（宏编辑器）**：
  - 手动添加移动、点击、双击、等待步骤。
  - **文字识别**：使用 Windows 原生 OCR（需 `winrt`）或 Tesseract（需安装引擎）截屏识别文字。
  - **图像匹配**：在指定区域搜索目标图片（需 `opencv-python`）。
  - **颜色检测**：检测区域内是否存在指定颜色。
  - **条件判断**：根据变量值（OCR结果、匹配坐标、颜色检测结果）实现跳转逻辑。
- **反检测机制**：
  - 独立控制连点器、回放、流程是否启用反检测。
  - 时间抖动：每次操作间隔随机增减（0~20%）。
  - 位置抖动：每次点击或移动坐标随机偏移（0~10像素）。
- **DPI 感知**：自动处理系统缩放，确保坐标准确。
- **全局热键**：
  - `F6`：开始/停止连点
  - `F9`：开始/停止录制
  - `F10`：回放录制
